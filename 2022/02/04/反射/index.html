# 反射

# 1. 为什么要使用反射？

当在写通用工具类时，不知道对象的真实类型，无法强转，强转有风险。

# 2. 获取字节码的三种方式

1.Class.forName()静态方法
2.对象.getclass方法
3.对象.class

# 3. 创建对象的方法

1、通过class对象的newInstance()方法
2、通过constructor对象的newInstance()

# 4. 获取方法的方法

1.获取自身和父类的构造方法(不包括私有的)
 Constructor[]constructors = clz.getconstructors();
2.获取自身的公有和私有的构造方法
 Constructor[] declaredconstructors =clz.getDeclaredConstructor();
3.获取自身和父类的公有方法(不包括私有的)
 Method[ ]methods =clz.getMethods();
4.获取自身的公有和私有方法
 Method[]declaredMethods =clz.getDeclaredMethods()

# 5. 方法访问权限设置

.setAccessible(true) 设置访问权限，将private修饰的方法变为public

# 6. 方法调用

方法.invoke(Object obj, Object... args) 调用方法(前面是对象，后面是需要的参数)
